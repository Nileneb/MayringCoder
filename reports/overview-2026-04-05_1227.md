2. `services` - Styles for the services section, which may display a list of services offered by the website.
3. `cases` - Styles for the cases section, which may showcase past projects or examples of work done by the website owner.
4. `process` - Styles for the process section, which outlines the steps involved in working with the website owner.
5. `split-grid` - Styles for a grid layout where elements are split into two columns.
6. `nav-links` and `nav-cta` - Navigation links and call-to-action styles.
7. `contact-form` - Styles for the contact form, including input fields, labels, and submission button.
8. `footer` - Styles for the footer section of the website, which typically includes links, social media icons, and copyright information.
9. Media queries - These allow the styles to adapt based on the size of the user's screen, ensuring that the layout looks good on both desktop and mobile devices.
