3. **Layout**: Utilizes CSS Grid and Flexbox for structuring the page, creating grids (e.g., services grid, cases grid), and defining the general layout of sections like the hero section, footer, and navigation menu.

4. **Interactive Elements**: Styles form elements like input fields, buttons, and dropdown menus. Also includes hover and focus states to enhance user experience.
